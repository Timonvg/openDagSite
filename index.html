Wat ga ik hier leren?

Je leert hier van alles. Als eerste begin je met simpele blokjes code bij elkaar te slepen, maar al snel begin je echt code te schrijven. Binnen een half jaar kan je al best veel! Je gaat in het eerste jaar vooral standaard web-development leren en de programmeertalen Python/C#.

Welke vakken krijg ik allemaal?

PRA

Hier leer je echt vooral in groepjes werken, en ga je met kleine groepjes grote applicaties/ websites maken. Teamwork is heel belangrijk hier.

PRO

Hier leer je eigenlijk alles dat te maken heeft met programmeren, zonder het programmeer gedeelte. Je leert dus plannen, ook leer je hoe je in groepjes werkt.

WEB

WEB is precies wat je denkt dat het is. In deze lessen leer je vooral web-development. Je leert HTML code, styling, en ook de code voor het brein van je site.

NATIVE

Hier leer je vooral echt software programmeren. Eerst start je met de simpele programmeertaal Python, en later leer je ook echt hele programma's maken!

Generiek

Programmeren is natuurlijk heel leuk, maar je hebt ook nog gewone vakken zoals Engels, Nederlands en Rekenen. Hier krijg je vooral herhaling van wat je al hebt geleerd op school.

DIV

DIV staat voor Digitale Vaardigheden. Hier leer je vooral meer met programma's zoals Excel en Word omgaan, en leer je veiliger te zijn online.
